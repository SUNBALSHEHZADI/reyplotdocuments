# 📘 ReyPlot Documentation

[![MkDocs](https://img.shields.io/badge/MkDocs-Material-526CFE?logo=materialformkdocs&logoColor=white)](https://www.mkdocs.org/)
[![GitHub license](https://img.shields.io/github/license/SUNBALSHEHZADI/reyplot_Documents)](LICENSE)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](https://github.com/MuhammadEssa2002/reyplot/pulls)

This repository contains the **complete MkDocs documentation** for [ReyPlot](https://github.com/MuhammadEssa2002/reyplot) a lightweight 2D scientific plotting library built in Python.

The documentation is written with the **Material for MkDocs** theme and includes detailed user guides, API references, code examples, and visual galleries.

---

## 🌟 Features of This Documentation

- ✅ **Professional layout** – clean, responsive, searchable
- ✅ **Comprehensive coverage** – installation, quick start, scatter plots, axes, layers, block grid, background images, titles, legend, saving, limits, API reference, colors, and examples
- ✅ **Code‑first approach** – every feature includes copy‑ready code blocks
- ✅ **Parameter tables** – clear descriptions of all function arguments
- ✅ **Gallery of examples** – real‑world use cases with images
- ✅ **Light/dark theme** – automatic switching with Material theme
