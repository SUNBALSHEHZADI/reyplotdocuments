# **Layers**

ReyPlot uses layers to build the plot background. You can customize the inner and outer layers with colors and gradients.

## **Inner Layer**

The inner layer is the background of the plotting area (the region inside the axes). Use `fig.inner_layer()`:

```python
fig.inner_layer(color="#EEEEEE", gradient=False, gradient_color="#000000", alpha=1)
```
